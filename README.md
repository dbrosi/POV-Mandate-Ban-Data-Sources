# POV-Mandate-Ban-Data-Sources

Provided here are the data used to track changes in county-level, weekly, 1<sup>st</sup>-dose COVID-19-vaccinations before and after proof-of-vaccine (POV) mandate ban compared with counties within states that did not enact a POV mandate ban. Date on POV mandate ban enactment date are available at the National Academy for State Health Policy (NASHP) website. 

The main data source used to calcualte weekly 1<sup>st</sup>-dose COVID-19 vaccinations are from the Center's for Disease Control and Prevention's (CDC) daily COVID-19 vaccination data (COVID-19_Vaccinations_in_the_United_States_County.7z), supplemented with state health department data from Texas for missing countly-level 1st-dose COVID-19 vaccination data (TX_Vaccination_Data.7z).
