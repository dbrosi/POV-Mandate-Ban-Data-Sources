# POV-Mandate-Ban-Data-Sources

Provided here are the data used to track changes in weekly 1st-dose COVID-19 

The main data source used to calcualte weekly 1<sup>st</sup>-dose COVID-19 vaccinations are from the Center's for Disease Control and Prevention's (CDC) daily COVID-19 vaccination data, supplemented with state health department data from Texas for missing countly-level 1st-dose COVID-19 vaccination data.
