# POV-Mandate-Ban-Data-Sources

Provided here are the data used to track changes in county-level, weekly, 1<sup>st</sup>-dose COVID-19-vaccinations before and after proof-of-vaccine (POV) mandate ban compared with counties within states that did not enact a POV mandate ban. Date on POV mandate ban enactment date are available at the National Academy for State Health Policy (NASHP) [website](https://www.nashp.org/state-lawmakers-submit-bills-to-ban-employer-vaccine-mandates/). 

The main data source used to calculate weekly 1<sup>st</sup>-dose COVID-19 vaccinations was the Center's for Disease Control and Prevention's (CDC) COVID-19 Vaccinations in the United States, County | Data (COVID-19_Vaccinations_in_the_United_States_County.7z), supplemented with state health department data from Texas for missing countly-level 1st-dose COVID-19 vaccination data (TX_Vaccination_Data.7z). 

Other data for covariates include the American Community Survey (ACS), Area Health Resouce File, county-level 2020 Election Results, John's Hopkins county-level data on COVID-19 cases and death, and county centroid distances (Census_AHRF_JH.7z) along with datasets detailing other state and local COVID-19 vaccination policy (COVID-19_Policy_Data.7z).
